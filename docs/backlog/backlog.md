# Smart Data Planner - Product Backlog

## High Priority Features

1. **Task Management Core**: The ability to add tasks & their information to a to-do list
2. **Multiple Lists**: Creating and managing multiple to-do lists (Categories)
3. **Time Scheduling**: Schedule times during each day to work on tasks during a week
4. **Calendar Integration**: Integrate scheduled tasks around existing events and/or classes during a day
5. **Automatic Scheduling**: Tasks can automatically be added to the user's schedule, helping plan the user's time efficiently
6. **Simple and Beautiful UI/UX**: Intuitive and visually appealing user interface
7. **Task Deletion**: Enables users to dispose of tasks they don't need
8. **Reminder Settings**: Adjustable amount of days for reminders

## Medium Priority Features

1. **Drag & Drop**: Drag tasks to calendar
2. **Categorization & Prioritization**: Categorize/tag reminders/todos and label priority levels
3. **Automatic Planning Rules**: Define rules for automatic planning (set times when tasks can be scheduled)
4. **Theme Toggle**: Enables users to change the app from light mode to dark mode
5. **Personalization**: Enables users to personalize the background color/image of their task manager

## Low Priority Features

1. **List Merging**: Combining to-do lists
2. **Functional API**: Provide a robust API for power users and integrations
3. **Task Suggestions**: Enables users to pick from a list of pre-suggested tasks (by the app)
4. **Push Notifications**: Enables users to turn on push notifications (text message) for their tasks

## High Priority User Stories
1. As a user I want to remove tasks from my to-do lists.
2. As a user I want to access removed tasks before they are deleted.
3. As a user I want to create tasks and add them to a to-do lists so that I can keep track of things I have to get done during the week.
4. As a user I want to edit existing tasks so they're they align with my changing goals.
5. As a user, I want the planner to look visually appealing and simple to use so that I will want to use it more and be productive.
6. As a user I want my tasks to automatically be scheduled around tasks in my calendar so that I do double book.
7. As a user I want to be able to choose to automatically add tasks so that my planning is more efficient.
8. As a user I want be able to adjust the amount of days for a reminder to go off so that I can hold myself accountable better.


## Medium Priority User Stories
1. As a user I want to restrict what kind of tasks I can set during a chosen time interval so that I can be more organized with my planning.
2. As a user I want to be able to choose the background on the planner so that my planner can look appealing and be easy to navigate task types.
3. As a user I want to be able to group my tasks together/reminders so that i can be more organized
4. As a user I want to be able to prioritize my tasks/reminders so that I will do the most important tasks first.
5. As a user I want to be able to change from light mode to dark mode so that my planner has more personalization.

## Low Priotity User Stories
1. As a user, I want to pick from a list of pre-suggested tasks so that I can remember certain tasks that I may have forgotten, or find tasks that I desire to complete.
2. As a user, I want to be notified of upcoming tasks so that I can complete them.
3. As a user, I want to merge my task lists so that I can better manage my app storage and organization.
4. As a power user, I want a robust API so that it can integrate seamlessly with my workflow.
