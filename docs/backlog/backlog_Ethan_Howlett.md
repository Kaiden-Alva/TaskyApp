# Ethan Howlett - Product Backlog

## Features

1. **Automatic Scheduling**: Tasks can automatically be added to the user's schedule, helping plan the user's time efficiently.
2. **Categorization & Prioritization**: Categorize/tag reminders/todos and label priority levels.
3. **Functional API**: Provide a robust API for power users and integrations.
4. **Simple and Beautiful UI/UX**: Intuitive and visually appealing user interface.
5. **Automatic Planning Rules**: Define rules for automatic planning (set times when tasks can be scheduled).
